The live site is generated at the repository root as a single English index.html. Edit config/site.json (and optional English strings under locales/messages.json in strings.en, or use site.json content / contentLocales.en), add images under assets/, then run py tools/render_static_pages.py (or push to GitHub Actions). Visitors can use the browser or OS translate feature for other languages. Do not copy this file to the site root.
